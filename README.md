## できること

- アイドル名・色合いから個人カラーを検索
- RGB / HSV / HEX 形式でカラーをコピー
- よく使う・お気に入りのカラーを Keep

## Thanks!

アイドルの色情報は [im@sparql](https://sparql.crssnky.xyz/imas/) より取得したデータを利用しています。
